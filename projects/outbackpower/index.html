Weekly Workflow Monday → Friday: who does what, when

▼

● Monday — Team Leads Update

1
Open your assigned Excel file (received via Dropbox or email)
2
Update only these 4 columns for each task:

Status

Not Started • In Progress • Complete • At Risk • Blocked

% Complete

Enter 0–100 to reflect current progress

Notes / Blockers

Current issues, progress notes, or blockers

Week Updated

Today’s date so PM knows it’s fresh

3
To add new tasks, use the empty rows at the bottom. Follow the Task ID pattern (e.g. PE-020, SC-018).
4
Save and return the file via Dropbox or email

● Tuesday — PM Reviews & Loads Gantt

1
Collect all returned Excel files from team leads
2
Open Outback_Power_Gantt_Chart.html in Chrome
3
Drag & drop all .xlsx files onto the drop zone (or click “Choose Files”)
4
Colored chips confirm which teams are loaded. The Change Log tracks every upload.
5
Review, filter, search, and print as needed

● Wednesday — Internal Review

1
Open PM_Console.xlsx → update the Dashboard and Weekly Review Notes

● Thursday / Friday — Customer Update

1
Open PM_Console.xlsx → “Customer Status Report” tab → fill in weekly summary → share with Outback Power

Gantt Chart Features What this tool can do

▼

📦 Drag & Drop Upload

Drop multiple .xlsx files at once. The tool auto-detects which team each file belongs to.

✅ Team Status Chips

9 colored chips show which files are loaded and when they were last uploaded.

🔍 Filter & Search

Filter tasks by status or team. Search for any task by keyword.

🔎 Zoom Levels

Switch between weekly, monthly, and quarterly views for different planning horizons.

✏️ Inline Editing

Click any Gantt bar to edit status and % complete directly in the chart.

📅 Today Marker

A red vertical line shows the current date on the timeline.

📄 Change Log

Every upload is recorded with timestamp, task count, and data hash for audit trail.

🖨 Print & Export

Print the Gantt chart for meetings or export task data as CSV.

🔴

Status Definitions What each status color means

▼
Not Started — Work has not begun on this task
In Progress — Actively being worked on
Complete — Task is finished and verified
At Risk — May miss target date or has emerging issues. Flag early!
Blocked — Cannot proceed due to a dependency or external issue
🔒

Document Control & Security Sheet protection, passwords, and audit trail

▼

All team Excel files are sheet-protected to prevent accidental changes to headers, formulas, and structure. Team leads can only edit the 4 update columns (Status, % Complete, Notes, Week Updated) and the empty rows at the bottom for new tasks.

🔒 Sheet Protection Password: OutbackPM2026

Share this password only with people who need to modify the file structure (e.g., add columns, edit headers). Regular team leads do not need the password — their editable cells are already unlocked.

💡The Change Log (visible on the Gantt Chart tab) records every file upload with timestamp, task count, and data hash — so you always know who submitted what and when.
💡

Tips & Best Practices Get the most out of this tool

▼
📋Only edit your assigned file. Don’t rename columns or restructure sheets — the Gantt chart depends on the exact column layout.
➕Adding tasks: Use the empty rows at the bottom. Follow the Task ID pattern for your team (e.g. PE-020, SC-018, BMS-013).
🌐Works offline after first load — the SheetJS library caches in your browser. No internet required to load Excel files.
⚠️Use “At Risk” early — flag tasks before they become “Blocked.” Early warnings give the PM time to help resolve issues.
📩Dropbox or email — either works for file delivery. Just make sure the filename stays the same so the Gantt chart can detect the team.
🚀Use Chrome for best results. The Gantt chart uses modern CSS features that work best in Chrome or Edge.
